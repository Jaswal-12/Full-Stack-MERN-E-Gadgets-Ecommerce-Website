# 🛒 E-Gadgets Ecommerce Website

A full-stack MERN ecommerce website built using React, Node.js, Express, and MongoDB.

---

## 🚀 Features

- User Authentication (Login/Register)
- Product Listing
- Add to Cart
- Cart Management
- Responsive UI
- Backend API Integration
- MongoDB Database
- Context API / State Management

---

## 🛠 Tech Stack

### Frontend
- React.js
- CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
